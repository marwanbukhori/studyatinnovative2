# Study At Innovative — v2

Conversion-focused landing page for Diploma Business Administration (ODL) with Dana Pendidikan Inovatif (DPI) bursary.

**Stack:** Vue 3 + Vite + Three.js

## Dev

```bash
npm install
npm run dev       # http://localhost:5174
npm run build     # outputs to /dist
npm run preview
```

## Structure

- `src/App.vue` — all sections in one file
- `src/components/HeroCanvas.vue` — Three.js hero (desktop + lite mobile)
- `src/components/LogoIUC.vue`, `Icons.vue` — brand + icon set
- `public/assets/innovative.png` — official IUC wordmark
- `public/{robots.txt,sitemap.xml,favicon.*}` — SEO assets
